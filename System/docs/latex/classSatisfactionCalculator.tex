\doxysubsection{Detailed Description}
\mbox{\hyperlink{classCommand}{Command}} class responsible for calculating the satisfaction level of the population. 

The \mbox{\hyperlink{classSatisfactionCalculator}{Satisfaction\+Calculator}} class inherits from \mbox{\hyperlink{classCommand}{Command}} and uses an \mbox{\hyperlink{classSCReceiver}{SCReceiver}} to calculate satisfaction based on factors like tax type, population, and map data.
