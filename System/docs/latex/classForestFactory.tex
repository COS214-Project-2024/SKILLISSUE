\doxysubsection{Detailed Description}
Factory class responsible for creating \mbox{\hyperlink{classForest}{Forest}} tiles. 

This class inherits from \mbox{\hyperlink{classTileFactory}{Tile\+Factory}} and provides a specific implementation for creating tiles of type \mbox{\hyperlink{classForest}{Forest}}, using a texture and animations as input.
