\doxysubsection{Detailed Description}
Abstract base factory class for creating \mbox{\hyperlink{classTile}{Tile}} objects. 

The \mbox{\hyperlink{classTileFactory}{Tile\+Factory}} class defines the framework for tile creation, following the factory pattern, allowing derived factories to create various types of tiles.
