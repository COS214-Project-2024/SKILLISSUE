\doxysubsection{Detailed Description}
Factory class for creating \mbox{\hyperlink{classRoad}{Road}} tiles. 

The \mbox{\hyperlink{classRoadFactory}{Road\+Factory}} class follows the factory pattern, providing methods to instantiate \mbox{\hyperlink{classRoad}{Road}} tiles with specified textures and animations.
