\doxysubsection{Detailed Description}
A concrete tax policy representing a high tax rate. 

The \mbox{\hyperlink{classHighTax}{High\+Tax}} class inherits from \mbox{\hyperlink{classTaxPolicy}{Tax\+Policy}} and provides an implementation for calculating tax based on a high tax rate, affecting the revenue accordingly.
