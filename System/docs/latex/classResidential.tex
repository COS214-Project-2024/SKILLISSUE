\doxysubsection{Detailed Description}
Represents a residential tile in the simulation. 

The \mbox{\hyperlink{classResidential}{Residential}} class inherits from \mbox{\hyperlink{classTile}{Tile}} and defines properties and behaviors specific to residential zones, where citizens reside within the simulation.
