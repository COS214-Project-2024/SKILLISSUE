\doxysubsection{Detailed Description}
Represents a map composed of tiles within the simulation. 

The \mbox{\hyperlink{classMap}{Map}} class manages the grid of tiles, supports selection, drawing, loading, and saving, and performs operations to manage connectivity and direction of various tiles.
