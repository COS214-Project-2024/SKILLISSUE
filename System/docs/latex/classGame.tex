\doxysubsection{Detailed Description}
Manages the main game loop, state stack, and resources. 

The \mbox{\hyperlink{classGame}{Game}} class initializes and manages the game window, resources like textures and tiles, game states, and factories for tile creation. It provides methods to control game states, load assets, and run the main game loop.
