\doxysubsection{Detailed Description}
Handles the distribution of resources across the map. 

The \mbox{\hyperlink{classDRReceiver}{DRReceiver}} class provides the functionality to distribute resources to various tiles on the map, interacting with the \mbox{\hyperlink{classResourceManager}{Resource\+Manager}} to manage resources.
