\doxysubsection{Detailed Description}
Factory class for creating \mbox{\hyperlink{classWater}{Water}} tiles. 

The \mbox{\hyperlink{classWaterFactory}{Water\+Factory}} class follows the factory pattern, providing methods to instantiate \mbox{\hyperlink{classWater}{Water}} tiles with specified textures and animations.
