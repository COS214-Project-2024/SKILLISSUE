\doxysubsection{Detailed Description}
An abstract interface for resource management operations. 

The \mbox{\hyperlink{classResourceInterface}{Resource\+Interface}} class defines a set of pure virtual methods that must be implemented by any class responsible for managing resources such as water, power, sewage, and material. This interface ensures a consistent API for resource management across different implementations.
