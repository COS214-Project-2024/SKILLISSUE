\doxysubsection{Detailed Description}

The \mbox{\hyperlink{classDistributeResources}{Distribute\+Resources}} class is responsible for managing the distribution of resources by interacting with a \mbox{\hyperlink{classDRReceiver}{DRReceiver}} and updating the map as needed.
