\doxysubsection{Detailed Description}
Abstract base class representing a state in the game. 

The \mbox{\hyperlink{classGameState}{Game\+State}} class provides a blueprint for specific game states by defining common methods for drawing, updating, and handling input. Each state is associated with a {\ttfamily \mbox{\hyperlink{classGame}{Game}}} instance and must implement the abstract methods defined here.
