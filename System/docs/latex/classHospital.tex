\doxysubsection{Detailed Description}
Represents a hospital tile in the simulation. 

The \mbox{\hyperlink{classHospital}{Hospital}} class inherits from \mbox{\hyperlink{classTile}{Tile}} and is used to define properties and behaviors specific to hospital tiles, which are responsible for health-\/related services within the simulation.
