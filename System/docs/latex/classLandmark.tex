\doxysubsection{Detailed Description}
Represents a landmark tile in the simulation. 

The \mbox{\hyperlink{classLandmark}{Landmark}} class inherits from \mbox{\hyperlink{classTile}{Tile}} and is used to define properties and behaviors specific to landmark tiles, which may contribute to aesthetics, tourism, or city appeal.
