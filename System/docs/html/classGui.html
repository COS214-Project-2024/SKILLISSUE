Detailed Description

Represents a graphical user interface (GUI) element for menus and interactive lists.

The Gui class provides a flexible GUI system for displaying a menu of entries, either horizontally or vertically. Each entry can be customized with text and styling, and users can interact with entries to perform various actions.
