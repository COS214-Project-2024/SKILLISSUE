Detailed Description

Represents a single entry within a graphical user interface (GUI) component.

A GuiEntry consists of a visual shape, a message string, and text displayed on the entry. It is designed to be used as part of a larger GUI, such as a menu or list, where each entry can display a label and carry an associated action or message.
