\doxysubsection{Detailed Description}

The \mbox{\hyperlink{classIndustrialFactory}{Industrial\+Factory}} class follows the factory pattern, providing methods to instantiate \mbox{\hyperlink{classIndustrial}{Industrial}} tiles with specified textures and animations.
