\doxysubsection{Detailed Description}
Abstract base class for tax policy implementations. 

The \mbox{\hyperlink{classTaxPolicy}{Tax\+Policy}} class provides a framework for different types of tax policies, defining methods for calculating tax and retrieving policy details.
